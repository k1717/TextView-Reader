# GitHub Upload Notes

This folder is a cleaned source-code package for GitHub.

Excluded intentionally:

- `.gradle/`
- `build/`
- `app/build/`
- `.idea/`
- `local.properties`
- APK/AAB files
- signing keys (`*.jks`, `*.keystore`)

To upload through the GitHub webpage, unzip this package and upload the contents of this folder to the repository.

Do not upload `local.properties`; Android Studio recreates it automatically with your local SDK path.
